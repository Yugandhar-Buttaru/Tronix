# Tronix - TRON-Inspired Lightcycle Game

A futuristic TRON-inspired lightcycle racing game built with HTML, CSS, and JavaScript, featuring neon visuals, AI opponents, and engaging gameplay mechanics.

## 🎮 Features

### Core Gameplay
- **Neon TRON Arena** - Glowing grid-based battlefield
- **Light Trail Mechanics** - Deadly trails that eliminate players on contact
- **AI Opponents** - Computer-controlled lightcycles with different colors
- **Collision Detection** - Advanced system for walls, self-collision, and opponent trails
- **Boost System** - Temporary speed increase with visual and audio feedback

### Visual Effects
- **Neon Glow Effects** - Authentic TRON aesthetic with canvas shadows
- **Particle System** - Boost effects and collision explosions
- **Enhanced Trails** - Thick, rounded lines with gradient colors
- **Modern UI** - Animated buttons, gradients, and smooth transitions

### Audio System
- **Web Audio API** - Engine sounds, boost effects, collision impacts
- **Game Over Melody** - Dramatic ending sequence
- **Dynamic Feedback** - Audio cues for all major actions

### Game Systems
- **Score Tracking** - Points for survival + 100 bonus for eliminating opponents
- **High Score System** - Persistent across game sessions
- **Game States** - Menu, playing, paused, game over with proper transitions
- **Responsive Design** - Adapts to different screen sizes

## 🎯 Controls

- **W/A/S/D** - Movement (Up/Left/Down/Right)
- **Shift** - Boost (temporary speed increase)
- **Escape** - Pause/Resume game
- **Space** - Start game / Restart after game over

## 🚀 How to Play

1. **Survive** by avoiding walls and all light trails
2. **Eliminate opponents** by forcing them into collisions with trails or walls
3. **Use boost strategically** for escapes or aggressive moves
4. **Compete for high scores** by surviving longer and eliminating opponents

- **HTML5 Canvas** - Game rendering and graphics

## 🚀 Getting Started

1. **Clone or download** the project files
2. **Open `index.html`** in a modern web browser
3. **Click "START GAME"** or press Space to begin
4. **Use keyboard controls** to play the game

## 🎮 Game Strategy Tips

- **Corner opponents** by cutting off their escape routes
- **Use walls** strategically to trap AI opponents
- **Save boost** for emergency escapes or aggressive attacks
- **Watch opponent patterns** - AI follows predictable movement
- **Control the center** of the arena for maximum maneuverability

## 🏆 Scoring System

- **1 point** per frame survived while alive
- **100 bonus points** for each opponent eliminated
- **High score tracking** across all game sessions

## 🐛 Known Issues

- Audio may not work in some browsers until user interaction
- Performance may vary on older devices with many particles
- AI opponents use simple pattern-based movement
