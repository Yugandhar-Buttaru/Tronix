**Game Name :**

Tronix

**README Description :**

Tronix is a futuristic TRON-inspired lightcycle racing game built with HTML, CSS, and JavaScript, featuring neon visuals, boost mechanics, and interactive controls in a glowing grid arena.

**Game folder :** 

tron-race-game
│
├── index.html
├── style.css
├── script.js
└──Readme.md

**Features :** 

- Neon TRON arena
- Light trail behind bike
- Trail collision detection
- Wall collision detection
- Boost ability
- Score counter
- Start screen
- Restart button
- Grid arena

**Control :**

W → Forward
S → Backward
A → Left
D → Right
Shift → Boost
